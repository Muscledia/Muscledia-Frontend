Feature Request: Workout Completion Analytics Display
Overview
When a user completes a workout, display comprehensive analytics including performance metrics, personal records achieved, and workout summary.

User Story
As a user
I want to see my workout performance metrics immediately after completing a workout
So that I can track my progress and celebrate achievements

Acceptance Criteria
1. Analytics Display Trigger

Show analytics screen/modal automatically when workout status changes to COMPLETED

User can access analytics from workout detail screen

Analytics visible for both in-progress and completed workouts
2. Metrics to Display
Core Metrics:


Total Volume (kg) - metrics.totalVolume

Total Sets - metrics.totalSets

Total Reps - metrics.totalReps

Duration (minutes) - durationMinutes

Calories Burned - metrics.caloriesBurned
Personal Records:


Total PRs Achieved - metrics.personalRecordsAchieved (count)

PR Badges on Individual Sets - exercises[].sets[].personalRecords[]

Visual highlight for sets with PRs (e.g., badge, color, icon)
Muscle Groups:


Worked Muscle Groups - metrics.workedMuscleGroups[]

Endpoints:

GET /api/workouts/{workoutId} - Fetch workout details with metrics
PUT /api/workouts/{workoutId}/complete - Complete workout (triggers analytics)
5. User Experience
Flow:

User clicks "Finish Workout"
System processes completion
Analytics screen appears with:
Confetti animation (if PRs achieved)
Metrics summary
PR highlights
Share/Save options
User can:
View detailed breakdown
Share on social media
Return to workout list
Edge Cases:


Handle zero PRs gracefully (no PRs section shown)

Display "In Progress" metrics for active workouts

Show partial metrics if workout incomplete

API Documentation
Workout Response:

interface WorkoutResponse {
  id: string;
  status: 'IN_PROGRESS' | 'COMPLETED' | 'CANCELLED';
  durationMinutes: number | null;
  metrics: {
    totalVolume: number;
    totalSets: number;
    totalReps: number;
    caloriesBurned: number;
    workedMuscleGroups: string[];
    personalRecordsAchieved: number;  // Count of total PRs
  };
  exercises: Exercise[];
}

interface Exercise {
  sets: WorkoutSet[];
}

interface WorkoutSet {
  setNumber: number;
  weightKg: number;
  reps: number;
  personalRecords: string[] | null;  // ["MAX_WEIGHT", "MAX_VOLUME", etc.]
}
